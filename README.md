# Supercharged Jenkins docker image

Adds following components:

  * rsync
  * yum-utils
  * docker (for docker in docker access)
  * nodejs 6.x.x
  * bower
  * ruby
  * jekyll
  * maven 3.3.9
  * nuget
  * 7zip
  * gradle 2.14.1
  * wget
  * curl
  * debmirror
  * git
  * openjdk-8
  
https://hub.docker.com/r/huksley/jenkins/
